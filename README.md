# 📊 Power BI Project: YouTube Data Analysis

## 📌 Introduction
As a Power BI developer, the goal of this project is to analyze a YouTube dataset to extract valuable insights and provide actionable recommendations for content creators and marketers. The dataset consists of engagement metrics and video attributes sourced from YouTube's API, enabling analysis of viewer behavior, video performance, and factors influencing engagement.

This analysis helps stakeholders optimize content strategies and improve audience engagement on the platform.

---

## ❗ Problem Statement
The primary objective of this project is to uncover patterns, trends, and correlations within the YouTube dataset, focusing on engagement metrics and video attributes.

The analysis aims to answer key questions that assist content creators and marketers in:
- Improving video performance
- Increasing audience interaction
- Making data-driven content decisions

---

## 📂 Data Sources
- Data extracted using YouTube API
- ETL (Extract, Transform, Load) performed in Power BI
- Data handling follows YouTube’s terms of service and privacy guidelines

---

## 🛠️ Tasks Performed

### 📊 Data Analysis & Visualizations
1. Displayed **maximum uploads** using a card visual  
2. Displayed **minimum view ranks** using a card visual  
3. Displayed **average views for the last 30 days** (after data cleansing)  
4. Created a **line chart** highlighting titles with uploads exceeding 2 lakh  
5. Built a **spider (radar) chart** to show channel type vs. channel rank  
6. Developed a **packed bubble chart** for category-wise subscribers  
7. Created a calculated column:  
   👉 `Viewed by Subscriber = Views / Subscribers`  
8. Designed a **table visual** for channel type vs. views per subscriber  

---

### 📐 Advanced Power BI Features
9. Created a **measure for total uploads** and displayed using a card visual  
10. Built a **decomposition tree** for rank-wise category analysis  
11. Added **navigation buttons** for page switching  
12. Applied **formatting** with headings, themes, and logos  

---

### 🚀 Deployment & Sharing
13. Published the report to **Power BI Service**  
14. Created a **dashboard with key visuals**  
15. Exported report copies in **PDF / PPT format**  

---

## 📈 Key Insights (Example)
- High-performing channels show strong correlation between uploads and engagement  
- Certain content categories attract significantly more subscribers  
- Viewer engagement varies across channel types and rankings
